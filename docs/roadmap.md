# Roadmap

Progressive tasks to learn SwiftUI and iOS development together. Each task builds on the previous one.

## Tasks

- [x] **1. Project setup** — Create Xcode project, push to GitHub, both clone it
- [ ] **2. Basic button** — Replace the "Hello World" with a big red button that prints to console
- [ ] **3. Play sound on tap** — Use AVFoundation to play jarabe_tapatio.mp3 when the button is tapped
- [ ] **4. Button animation** — Add a scale or rotation animation when the button is tapped
- [ ] **5. Custom button style** — Add colors, fonts, images, shadows to make the button look great
- [ ] **6. Multiple sounds** — Add more sound files and play a random one on each tap
- [ ] **7. Haptic feedback** — Make the phone vibrate when the button is tapped
- [ ] **8. Settings screen** — Add a NavigationStack with a settings screen (volume, sound selection)
- [ ] **9. Dark mode** — Support dark mode with appropriate colors
- [ ] **10. Launch screen** — Add a custom launch screen with mariachi theming
